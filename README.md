# HIDROFLOW

Proyecto HIDROFLOW – Plataforma hidrológica senior basada en **Vite + React + SIATA + GT-AS-004**.  
Incluye una arquitectura profesional con CI/CD, Release Drafter, proxy opcional para SIATA y estructura escalable para futuros módulos hidrológicos.

---

## 🚀 Tecnologías principales
- **React 18 + Vite 5**
- **Proxy Node/Express** para SIATA (opcional)
- **GitHub Actions** (CI)
- **Release Drafter** (automatiza changelog)
- **Convenciones SemVer** (versionado profesional)
